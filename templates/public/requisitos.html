Requisitos

El workshop se realizara el día 10 de Junio, desde las 15:00, y el sábado 11 de Junio, desde las 9:00 AM

Una vez confirmada la gente debe verificar los siguientes contenidos para el sábado.

Cada persona debe llevar una identificación válida para poder ingresar al evento.

Se debe llevar idealmente notebook para trabajar, pero no es un requisito, cada equipo, de a lo más 5 personas, debería poder trabajar con 2 o 3 computadores.

El juego debe desarrollarse durante el día, al participar se da por conocida esta regla, los que no la cumplan serán descalificados.

Los equipos pueden llegar formados o formarse en el evento, pero deben ser de a lo más 5 personas y no menos de 3, esto sin contar a profesores y mentores apoyando.

A la gente que va a llevar notebook le recomendamos instalar el siguiente Software que puede serles de utilidad para el desarrollo de video juegos en 1 día:

  • Blender es un software open source de modelado 3D que además incluye un motor de video juegos que permite crear juegos sin la necesidad programar código o usando python, corre en Windows, Mac y Linux, descargaló aquí: blender
  • Gimp es un software de edición de imagenes 2D muy potente además de ser open source, se puede instalar en Windows, Mac y Linux: Gimp
  • Android es una plataforma móvil cada día más usada, para desarrollar basta saber programar en Java, no se requiere comprar software especial ni licencias, kit de desarrollo: Android SDK; Plataforma Eclipse:Eclipse (Eclipse requiere instalar un paquete adicional para desarrollo Android). Todo el software necesario para desarrollo de Android se puede usar desde Windows, Mac y Linux por lo menos.
  • andengine es un engine para desarrollo de juegos para Android muy usado, simple de entender y rápido para desarrollar: andengine
  • cocos2d es un framework para desarrollo rápido de juegos u otros medios interactivos, existe una versión con soporte para iPhone, actualmente activamente en desarrollo: Sitio Principal Sitio iPhone
